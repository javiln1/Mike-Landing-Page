# Remote Sales Academy - Landing Page

High-converting sales funnel for Mike's Remote Sales Academy program.

## Features

- Dark theme with red/black branding
- Wistia video integration
- Calendly booking widget
- Video testimonials
- Mobile-first responsive design
- UTM parameter tracking
- Formspree form integration

## Deployment

This site is deployed on Vercel and serves static files from the `output` directory.

### Local Development

```bash
python3 -m http.server 8000 -d output
```

Visit: http://localhost:8000

## Structure

- `output/` - Production-ready files
  - `index.html` - Main landing page
  - `education-page.html` - Resources page
  - `confirmation-page.html` - Booking confirmation
  - `assets/` - Images and media files
- `config/` - Configuration files
- `templates/` - HTML templates

## Technologies

- HTML5/CSS3
- Vanilla JavaScript
- Wistia (video hosting)
- Calendly (booking widget)
- Formspree (form handling)
